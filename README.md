# Lumière Hair Studio 💇

A professional, elegant, and artsy **hair salon website** — fully responsive for both web and mobile.

## ✨ Features
- **Hero section** with cinematic overlay
- **Services grid** with hover animations
- **Masonry-style gallery**
- **Booking / Registration form** with tabbed Sign In
- **Testimonials** with featured card highlight
- **Contact section** with embedded map
- **Mobile hamburger nav** fully functional
- **Scroll reveal** animations
- **Google Fonts**: Cormorant Garamond (serif) + Jost (sans-serif)
- Colour palette: Warm cream, sand, gold, dark charcoal

## 📂 Structure
```
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🚀 Live Preview
Open `index.html` in any browser, or host via [GitHub Pages](https://pages.github.com/).

## 🎨 Customisation
- Replace Unsplash image URLs in `index.html` with your own photos
- Update salon name, address, pricing and services
- Connect the booking form to a backend (e.g. Netlify Forms, EmailJS)
